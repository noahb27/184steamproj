# 184steamproj
Analysis of steam game price changes for Stat184 final project assignment

## Project Overview
This project examines historical Steam video game price distribution across genres from 2015-2024. We analyze pricing trends, genre-specific patterns, and market dynamics using R.

## Data
- Steam app ID lookup table (165,000+ apps) 
- Historical pricing data (2015-2024, quarterly observations) 
- Genre classifications based on game characteristics
